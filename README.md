A python based vcs system called `pit` that is similar to `git`.
features include:
- [x] init
- [x] add
- [x] commit
- [x] log
- [ ] status
- [ ] diff
  

## libraries used:
- pathlib
- argparse
- sys

# installation :
git clone [zaid-kamil/pit](https://github.com/zaid-kamil/pit) or just copy the pit.py
